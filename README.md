# 🧠 Interactive DSA AI Tutor

_A Generative AI-powered teaching assistant for Data Structures and Algorithms._

**Author:** Ayush Pandey  
**Program:** MCA (AI & Data Science), Graphic Era Deemed University  
**Focus Area:** Use of Generative AI in different domains / AI for Quality Education

## 📖 Project Overview

This mini-project is a fully interactive web application designed to act as a strict, analytical tutor for students studying complex Data Structures and Algorithms (DSA).

Unlike standard chatbots that simply provide the final code to a problem, this application uses **Google's Gemini 2.5 Flash** model combined with rigorous system instructions to guide users through line-by-line execution, time/space complexity analysis, and conceptual hints—mimicking the experience of a technical interview or a structured A2Z DSA sheet.

## 🚀 Technology Stack

- **Python 3** (Core Backend Logic)
- **Streamlit** (Web Interface & Session State Management)
- **Google GenAI SDK** (LLM Orchestration)
- **python-dotenv** (Environment Security)

## 🛠️ How to Run Locally

### 1. Prerequisites

You will need Python installed on your machine and a free Google Gemini API key.

- Get an API key here: [Google AI Studio](https://aistudio.google.com/app/apikey)

### 2. Setup the Environment

Clone or download this repository, then open a terminal in the project folder and run:

```bash
# Create a virtual environment
python -m venv .venv

# Activate the environment (Windows)
.\.venv\Scripts\activate

# Activate the environment (Mac/Linux)
source .venv/bin/activate
```
